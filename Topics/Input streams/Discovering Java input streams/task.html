What would be the output of the following Java code? 

import java.io.*; 
public class Main {
    public static void main(String[] args) {
        try {
            FileInputStream fis = new FileInputStream("file.txt");
            byte[] arr = new byte[3];
            fis.read(arr); 
            for(byte b : arr) { 
                System.out.print((char) b); 
            }
            fis.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
The 'file.txt' contains the string 'JavaStreams'.
